# Events

nexSys relies on [eventStream](../eventStream/introduction.mdx) as the backend event handler. nexSys provides a variety of events that users may tap into for their own packages and uses.

## AffGot

Triggered when any affliction is gained by the player. Returns nexSys Aff object.

## AffLost

Triggered when any affliction is lost by the player. Returns nexSys Aff object.

## \<affliction\>GotAffEvent

Triggered when a specific affliction is gained by the player. Replace `\<affliction\>` with the actual affliction name. Returns specific nexSys Aff object.

## \<affliction\>LostAffEvent

Triggered when a specific affliction is lost by the player. Replace `\<affliction\>` with the actual affliction name. Returns specific nexSys Aff object.

## DefGot

Triggered when any defense is gained by the player. Returns nexSys Def object.

## DefLost

Triggered when any defense is lost by the player. Returns nexSys Def object.

## \<defence\>GotDefEvent

Triggered when a specific defense is gained by the player. Replace `\<defence\>` with the actual defense name. Returns specific nexSys Def object.

## \<defence\>LostDefEvent

Triggered when a specific defense is lost by the player. Replace `\<defence\>` with the actual defense name. Returns specific nexSys Def object.

## BalanceGot

Triggered when balance is regained by the player.

## BalanceLost

Triggered when balance is lost by the player.

## \<balance\>GotBalEvent

Triggered when a specific balance state is regained by the player. Replace `\<balance\>` with the actual balance type.

## \<balance\>LostBalEvent

Triggered when a specific balance state is lost by the player. Replace `\<balance\>` with the actual balance type.

## eqBalRecoveredEvent

Triggered when both balance and equilibrium have returned to the player.

## SystemPaused

Triggered when the nexSys system is paused.

## SystemUnpaused

Triggered when the nexSys system is unpaused.

## SystemSlowModeOn

Triggered when the nexSys system slow mode is enabled.

## SystemSlowModeOff

Triggered when the nexSys system slow mode is disabled.

## PromptEvent

Triggered when a new prompt is received.

## RoomChangedEvent

Triggered when the player has moved to a different room.

## ItemAddedToRoom

Triggered when an item has been added to the current room. Returns GMCP item object.

## ItemAddedToInv

Triggered when an item has been added to the inventory. Returns GMCP item object.

## ItemRemovedFromRoom

Triggered when an item has been removed from the current room. Returns GMCP item object.

## ItemRemoveFromInv

Triggered when an item has been removed from the inventory. Returns GMCP item object.

## ItemUpdatedRoom

Triggered when an item in the room has been updated. Returns GMCP item object.

## ItemUpdatedInv

Triggered when an item in the inventory has been updated. Returns GMCP item object.

## ItemListForRoom

Triggered when the list of items in the current room has been updated. Returns array of GMCP item objects.

## ItemListForInv

Triggered when the list of items in the inventory has been updated. Returns array of GMCP item objects.

## TargetSetEvent

Returns string value of new (current) target.

## TargetChanged

Returns string value of new (current) target.

## ClassChanged

Returns object on class change:

```js
{
  old: string,
  new: string
}
```

## HealthUpdated

Returns object for player's health:

```js
{
  max: int,
  current: int,
  diff: int
}
```

## ManaUpdated

Returns object for player's mana:

```js
{
  max: int,
  current: int,
  diff: int
}
```

## RageUpdated

Returns object for player's rage:

```js
{
  max: int,
  current: int,
  diff: int
}
```
